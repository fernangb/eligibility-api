# Eligibility API

## API to validate eligibilities

## Tech stack
* Backend: NestJS (Node)
* Unit tests: Jest

## You get to do cool things, like:
* Get annual CO2 savings if customer is eligible
* Get inegibility reasons if something is invalid

## Install dependencies

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
